# MATLAB App Design - Interactive Photos App

Interactive Photos App is a MATLAB App Designer project developed as a final project in a MATLAB course.

The application allows users to interact with photos through a graphical interface and apply image-related operations in an intuitive way.

## Project Description

Interactive Photos App was developed as part of a MATLAB course.

The project combines MATLAB programming, App Designer, image processing concepts, and an interactive graphical user interface.

The application includes:

- Interactive photo display
- Image selection and loading
- User-friendly graphical interface
- Image processing actions
- MATLAB App Designer components
- Exported MATLAB code version
- Final project presentation

## Technologies Used

## How to Run

1. Clone the repository:

`git clone https://github.com/oferavioz/MATLAB_Appdesign_InteractivePhotosApp.git`

2. Open MATLAB.

3. Open the file:

`InteractivePhotosApp.mlapp`

4. Run the application from MATLAB App Designer.

Alternatively, open one of the MATLAB `.m` files and run it from the MATLAB environment.
